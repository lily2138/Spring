class = type 클래스는 타입이다.

라운드티 / 점퍼 / 반팔티

Type		생성자
Shirt 라운드티 = new shirt()
Shirt 점퍼 = new shirt()
Shirt 반팔티 = new shirt()

라운드티, 점퍼, 반팔티는 모두 인스턴스

--------------------------------------------------

default constructor 기본생성자
내가 생성자 정의 안해도, 무조건 기본생성자는 있다.
하나의 클래스에는 반드시 하나이상의 생성자가 존재한다.

--------------------------------------------------------

Construct 생성자
: 객체가 생성될 때마다 호출되는 것
명시적 생성자가 하는 일은 Field 초기화이다.
클래스에 아무런 생성자가 없다면 컴파일러가 기본 생성자는 넣어준다
